How to read this
Distance = similarity
Distance between companies = semantic similarity. Companies near each other describe their work in similar terms.
Projection · TF-IDF → UMAP
Built from each company's description + categories + vertical. TF-IDF features compressed from ~2000 dimensions to 3D via UMAP, preserving local structure (close pairs stay close).
Cluster halos
Soft volumes = emergent KMeans clusters in the TF-IDF space. The terms shown in the legend are the most distinctive words for each cluster.
Filaments
Curved tubes = pairs sharing one or more investors. Brighter = more shared investors.
Hover · nearest neighbors
Hovering a node draws beams to its top-5 nearest semantic neighbors (computed in the original TF-IDF space — not the 3D projection).
Caveat UMAP is a non-linear projection. Absolute distances are not metric — only local neighborhoods are reliable. Treat clusters as "these companies do similar things," not "distance = quantitative similarity score."
